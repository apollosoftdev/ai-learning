# AI Learning Hub

An educational web app covering core AI topics, designed as a resource for AI presentations and learning.

## Topics

- What is AI
- Machine Learning
- Neural Networks
- Natural Language Processing
- Computer Vision
- AI Levels
- AI Types
- Generative AI
- AI Ethics

## Tech Stack

- **Backend:** Node.js (vanilla HTTP server)
- **Frontend:** HTML, CSS, JavaScript

## Getting Started

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
